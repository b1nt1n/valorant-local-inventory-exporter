# Security Policy

## Reporting a Vulnerability

If you believe this project has a security issue, open a private report to the maintainer instead of posting secrets publicly.

## Sensitive Data

Never share:
- Riot access tokens
- Entitlements token
- PUUID / subject IDs
- Raw lockfile contents
- Logs that include authenticated API responses

## Safe Sharing Checklist

Before publishing logs or screenshots:
- Remove account identifiers
- Remove token-like strings
- Remove private inventory details if needed
